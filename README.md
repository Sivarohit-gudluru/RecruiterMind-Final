# 🚀 RecruiterMind – AI-Powered Hiring Automation System

> An intelligent end-to-end recruitment solution that automates resume screening, skill gap analysis, and candidate evaluation using AI.

---

## 🧠 Overview

RecruiterMind is an AI-driven platform designed to simplify and optimize the hiring process. It helps recruiters efficiently evaluate candidates by analyzing resumes against job descriptions and generating actionable insights.

---

## ⚙️ Key Features

- 📄 Resume Screening  
- 🔍 Gap Analysis  
- 📊 HireScore Generation  
- 🤖 AI Interview Module (Planned)  
- 🎤 Voice-Based Confidence Analysis (Planned)  
- 📤 HR Forwarding System  

---

## 🔗 System Workflow

Resume → Screening → Gap Analysis → HireScore → AI Interview → Final Score → HR Forwarding

---

## 🛠️ Tech Stack

Backend: Python, FastAPI, NLP, pdfplumber  
Frontend: React.js, Tailwind CSS  
Tools: Uvicorn, JavaScript  

---

## 🚀 How to Run

Backend:
cd backend
python -m venv venv
venv\Scripts\activate
pip install fastapi uvicorn pdfplumber nltk python-multipart
python -m uvicorn main:app --reload

Frontend:
cd client
npm install
npm run dev
